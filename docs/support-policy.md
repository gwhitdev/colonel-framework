# Support Policy

## Version Support

- Active support: latest `1.x` release.
- Older pre-1.0 versions are not guaranteed to receive fixes.

## Runtime Support

- Colonel targets Bun as the primary runtime.
- Keep Bun reasonably current when running or developing Colonel apps.

## Backward Compatibility

- Public API compatibility is maintained across the 1.x line.
- Breaking API changes require a major version bump.

## Issue Scope

- Bug reports for documented 1.x behavior are in scope.
- Requests for new major capabilities may be scheduled for later releases.
